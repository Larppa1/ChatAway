## About The Project

[![Product Name Screen Shot][product-screenshot]](https://example.com)

ChatAway is the first React web app I've ever made. Maybe a bit too complex to begin with, but I managed to make most of it without tutorials. Some things, like message queries from Firebase I had to check from a YouTube tutorial. The app was tested by around 10 people, and they gave nocited some things that had to be changed (disallow empty messages, alert if username is already in use etc.).

## Test via local repo

To get a local copy up and running follow these simple example steps.

### Prerequisites

Install latest version of npm
* npm
  ```sh
  npm install npm@latest -g
  ```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/Larppa1/ChatAway
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
